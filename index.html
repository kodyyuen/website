Photoshop Recreation

This project was an attempt at a recreation of photoshop. The program includes some of photoshop's features such as applying filters (blurring and sharpening) and transformations (greyscale and sepia) to images, downscaling images, and saving and loading images. Similar to photoshop, users are also able to manipulate layers of images. They are able to make any layer visible/invisible and apply any of the previous features to it. The program also supports the option to save and load images in ppm, jpg, and png. When saving and loading, the program is able to hold the state of the image. All of these features can be done with the photo being displayed live within the GUI that was designed for the program using the Swing framework. An example of something you can do in my program is: create two layers, apply greyscale to layer two and set layer one to invisible. The GUI would display layer one, then the greyscaled layer after layer one was made invisible. Then, you can save your image anywhere on your desktop in ppm, jpg, or png. Your image can be loaded back into the program by selecting the folder created when saving and the program would be able to load the image with the first layer invisible and the second layer greyscaled. This program was coded with object-oriented principles in mind.

maze

This project demonstrated how graph searching algorithms can solve a maze and helped me view mazes from a different point of view. Originally, I was lost on how I could code something that solves a maze by itself. When I started thinking of a maze as a graph of edges and nodes, the process became much clearer to me. I implemented deques, queues, and recursive lists to create my depth-first search and breadth-first search algorithms.
